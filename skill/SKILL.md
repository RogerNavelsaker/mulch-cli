# Mulch

Use this skill when you need durable expertise notes, conventions, or reusable domain knowledge stored in git.

Preferred commands:

```bash
mulch --help
mulch status
mulch list
```

Rules:
- Use mulch for durable expertise, not transient scratch notes.
- Keep entries concise and high-signal.
- Prefer updating existing expertise over duplicating it.
